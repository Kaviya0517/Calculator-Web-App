Simple Calculator Web App

 Project Overview
This project is a simple and interactive **Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations and demonstrates how front-end technologies can be used to build functional web applications.

The main goal of this project is to practice **DOM manipulation, event handling, and JavaScript logic**.


 Features
- Perform basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Clear the entire display using **AC**
- Delete the last entered character using **C**
- Toggle positive/negative numbers using **+/-**
- Real-time display updates
- Simple and clean user interface


 Technologies Used
- HTML – Structure of the calculator
- CSS – Styling and layout
- JavaScript – Logic and functionality

 How It Works
1. The calculator buttons are created using **HTML**.
2. **CSS** is used to design the layout and make the calculator visually appealing.
3. **JavaScript** handles the functionality:
   - Detects button clicks
   - Updates the display
   - Performs calculations using JavaScript expressions
   - Handles special operations like **AC, C, +/-, and =**
